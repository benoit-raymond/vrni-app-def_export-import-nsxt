## EXPORT vrni config

## IMPORT into NSX-T
#### via JSON Expression file

#### via automatic vrni translation (based on prevoiusly exported json file)
python3 export-app-def.py --appname Funbike  --verbose --script_mode IMPORT --nsxt_fqdn "nsx-t-fqdn" --nsxtuser admin --export_folder ./export/
